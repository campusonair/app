# CampusOnAir

Place `.env` to setup.

```
export REACT_APP_TITLE="Hello World"
export REACT_APP_DESCRIPTION="Hello"
export REACT_APP_SKYWAY_API_KEY="xxxx"
export REACT_APP_AUTH0_DOMAIN="xxxx"
export REACT_APP_AUTH0_CLIENT_ID="xxxx"
```

Run following command.

```
$ npm run i18n
$ npm run po2json
```
